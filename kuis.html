Soal Pilihan Ganda: Sistem Persamaan Linear Dua Variabel (SPLDV)

  1. Jika x + y = 10 dan 2x - y = 4, maka nilai x dan y adalah?
    A. x = 4, y = 6
    B. x = 5, y = 5
    C. x = 3, y = 7
    D. x = 6, y = 4
  2. Jika 3x + 2y = 18 dan x - y = 2, maka nilai x dan y adalah?
    A. x = 4, y = 3
    B. x = 5, y = 2
    C. x = 6, y = 0
    D. x = 3, y = 5
  3. Hasil penyelesaian SPLDV 4x + y = 19 dan 2x - y = 3 adalah?
    A. x = 4, y = 3
    B. x = 3, y = 4
    C. x = 5, y = 2
    D. x = 6, y = 1
  4. Jika x + 2y = 8 dan 3x - y = 9, maka hasilnya adalah?
    A. x = 3, y = 2
    B. x = 2, y = 3
    C. x = 4, y = 1
    D. x = 1, y = 4
  5. Sistem persamaan 3x + 5y = 21 dan 2x - y = 4 memiliki solusi x dan y sebagai?
    A. x = 4, y = 2
    B. x = 3, y = 3
    C. x = 2, y = 4
    D. x = 5, y = 1
